# Inventory Hub - Frontend

A modern, premium React-based dashboard for managing inventory items.

## Technology Stack
- **Framework**: React 18
- **Build Tool**: Vite 9
- **Styling**: Vanilla CSS
- **Icons**: Lucide React
- **API Client**: Axios

## Development Setup

### Prerequisites
- **Node.js**: Version 22+
- **npm**: Version 10+

### Installation
```bash
npm install
```

### Running Locally (Dev Mode)
```bash
npm run dev
```

### Building for Production
```bash
npm run build
```

## Environment Variables
The application uses environment variables for configuration:
- `VITE_API_URL`: The URL of the backend API.
